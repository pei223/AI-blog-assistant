## AI-blog-assistant
OpenAIを用いたブログ生成ツール

## Setup
```
npm i
```

## Build/start
```
npm run clean-build-start
```

## Watch renderer
以下を実行
```
npm run clean-build-start
```

Watchモードでrendererをビルド
```
npm run build:renderer -- --mode=development  --watch
```

## Production build
```
npm run prdbuild:win
npm run prdbuild:mac
```
